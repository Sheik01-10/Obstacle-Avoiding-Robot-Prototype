 *Arduino UNO
 *Ultrasonic Sensor (HC-SR04)
 *Motor Driver (L298N)
 *DC Motors with Wheels

 *Draft Board
 *Jumpers Wires
 *Servo Motor

 *Ultrasonic sensor for obstacle detection

 *Arduino based automation
